# 🇻🇳🇩🇪 Cơ cấu Tổ chức & Vận hành — Hội DHS Hamburg

Website giới thiệu cơ cấu tổ chức, quy trình hoạt động và thông tin liên hệ của **Ban chấp hành Khoá 14 — Hội Du học sinh Việt Nam tại Hamburg**.

## 🔗 Xem website

👉 [https://hungryharvey.github.io/dhs-hamburg/](https://hungryharvey.github.io/dhs-hamburg/)

## 📄 Nội dung

| Tab | Mô tả |
|-----|-------|
| **Sơ đồ tổ chức** | Big4, 2 mảng hoạt động, các ban & tiểu ban |
| **Quy trình sự kiện** | 4 giai đoạn từ định hướng đến vận hành |
| **Nhiệm vụ các ban** | Chi tiết công việc thành viên chính/phụ từng ban |
| **Liên hệ** | Email, Facebook, Instagram, TikTok |

## 🛠️ Cách chỉnh sửa

Website chỉ gồm **1 file duy nhất** là `index.html`. Không cần cài đặt gì thêm.

### Cập nhật tên người / thông tin:
1. Mở file `index.html`
2. Tìm tên cần sửa (Ctrl+F)
3. Thay đổi nội dung
4. Commit & push lên GitHub → website tự cập nhật sau 1–2 phút

### Xem trước trên máy:
Mở file `index.html` bằng trình duyệt (Chrome, Firefox...) là xem được ngay, không cần server.
